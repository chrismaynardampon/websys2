Key Terms in Web Development

External Style Sheet
A separate file containing CSS rules that define the presentation of HTML documents across multiple pages.
ID Selector
An identifier used in CSS to target and style a specific HTML element based on its unique ID attribute.
Inline Style
CSS applied directly within an HTML element's style attribute, affecting only that specific element.
List Item
An individual entry in an HTML list, which can be part of an ordered or unordered list.
Nested
Elements placed within another element, forming a hierarchical structure, common in HTML and CSS.
Ordered List
A type of HTML list where items are numbered sequentially, typically displayed with numbers or letters.
Relative Links
URLs that specify the location of a resource relative to the current location of the document.
Unordered Lists
A type of HTML list where items are bulleted or marked with icons, without a specific sequence order.
Version Control
A system that records changes to files over time, allowing you to recall specific versions later.
WYSIWYG
"What You See Is What You Get" - editors or interfaces that display content in a form closely resembling its appearance when printed or displayed as a finished product.
